# @guoyunhe/create-node

## Install

### Run directly

```bash
npx @guoyunhe/create-node
```

### Install on system

```bash
npm i -g @guoyunhe/create-node
create-node
```

### Install in project

```bash
npm i -D @guoyunhe/create-node
```

Add script entry:

```json
{
  "scripts": {
    "create-node": "create-node"
  }
}
```

Run:

```bash
npm run create-node
```

## Options

### `--help`

Show help

### `--version`

Show version
